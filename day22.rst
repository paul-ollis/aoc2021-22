Spliiting stuff
===============

Given an embedded rectange.
::

      @@@@@@@@@@
      @@@@@@++@@
      @@@@@@++@@
      @@@@@@@@@@
      @@@@@@@@@@
      @@@@@@@@@@

Slice along the X axis to give:
::

      @@@@@@  @@  @@
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@

The left and right rectanges are done, but either could be empty. Slice the
remainder along the Y axis.
::

              @@
      @@@@@@      @@
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@      @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@
              @@

            ++
      @@@@@@++@@

      @@@@@@@@@@

::

              ++
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@

The Y axis.
::

              ++
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@  ++  @@
      @@@@@@      @@
      @@@@@@  @@  @@
      @@@@@@  @@  @@
              @@
